Vendora

- Vendora is a modern, multi-tenant SaaS e-commerce platform built to help sellers and buyers connect seamlessly. It provides a fully-featured marketplace with separate roles for sellers, buyers, and admins, with a focus on scalability, security, and ease of use.

Features
*Seller Features

- Upload and manage products

- Track orders and sales

- Product approval workflow (admin moderation)

- Analytics dashboard

*Buyer Features

- Browse and search products

- Add products to wishlist

- Make purchases securely

- Personalized recommendations

*Admin Features

- Approve or reject products submitted by sellers

- Manage users (buyers & sellers)

- Monitor platform activity

- Configure global settings and pricing tiers

*Platform Features

- Multi-tenant architecture

- Role-based access control

- Server-side rendering for performance

- Responsive UI

Tech Stack

- Frontend: Next.js, Tailwind CSS, Framer Motion, React, Material UI

- Backend: Node.js, Next.js API Routes

- Database: MongoDB

- Authentication: NextAuth.js

- Hosting / Deployment: Vercel

- Other: REST API, JWT, ObjectId references for relational data
